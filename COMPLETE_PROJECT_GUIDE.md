# FactFeed: A Modern News Aggregator with AI-Powered Analysis

This guide outlines the complete architecture and implementation plan for building FactFeed, a sophisticated news platform that scrapes, analyzes, and presents news from various sources with a focus on clarity and discrepancy reporting.

## 🎯 Project Vision

The goal is to create a platform that feels like a premium news website. Instead of just showing raw data, it will present clustered news stories as single, coherent articles. The core feature is an AI-generated summary that combines information from multiple sources and explicitly highlights any factual discrepancies (e.g., differences in numbers, names, or locations) with clear source attribution.

### Core User Experience
- **Homepage**: A dynamic, modern news homepage featuring top stories, trending topics, and categorized news sections (e.g., World, Tech, Politics).
- **Unified Article Page**: Clicking a story leads to a single page that presents a unified, AI-generated summary from multiple sources.
- **Inline Discrepancies**: Factual differences are clearly marked within the text, like *"Prothom Alo reports a cost of ৳50,000, while The Daily Star states it was ৳75,000."*
- **Source Transparency**: Users can easily see the list of original source articles that were used to generate the summary.

## 🏗️ Architecture: A Unified & Extensible Approach

We will build a monolithic application where the Spring Boot backend serves the React frontend. This simplifies deployment and development. The architecture is designed to be modular and extensible, particularly for adding new data sources and tweaking AI parameters.

### The 4-Step Pipeline
The entire system revolves around a clean, sequential data pipeline:

1.  **Scrape**: News articles are scraped from configured sources (RSS, web scraping) and stored in a raw `articles` table.
2.  **Summarize**: Each new article gets an initial, individual summary generated by the AI. This summary is used for the next step.
3.  **Cluster**: Using the individual summaries, the AI identifies articles from different sources that cover the same event and groups them into a `SimilarityCluster`.
4.  **Finalize**: For each cluster, all original article content is sent to the AI one last time to generate a final, combined summary. This is where discrepancies are identified and formatted for display.

## 📊 Database Schema

The schema remains simple and focused on the core entities.

### Core Entity: `SimilarityCluster`
```sql
CREATE TABLE similarity_clusters (
    id BIGSERIAL PRIMARY KEY,
    article_ids JSONB NOT NULL,           -- Array of original article IDs in this cluster
```

### Component & Page Structure
- **`HomePage.tsx`**: The main landing page.
  - A "hero" section for the most important story cluster.
  - Grids or lists for other story clusters, organized by categories (e.g., "World News", "Technology").
  - Each item on the homepage is a `ClusterCard`.
- **`ClusterCard.tsx`**: A preview card for a story cluster.
  - Shows an AI-generated headline for the cluster.
  - May show a key image from one of the source articles.
  - Lists the logos of the news sources in the cluster (e.g., BBC, Reuters, CNN).
- **`ArticlePage.tsx`**: The page for a single story cluster.
  - Displays the `final_summary` as the main article body.
  - Discrepancies are highlighted visually (e.g., different color, bold text, or an icon).

- **`CategoryPage.tsx`**: Shows all story clusters for a specific category (e.g., `/world`).

   `src/main/java/com/factfeed/scraper/BanglaWebScraper.java`:
   ```java
   @Component
   public class BanglaWebScraper implements Scraper {
       
       @Value("${factfeed.scraping.delay-between-requests:2000}")
       private int delayBetweenRequests;
       
       @Value("${factfeed.scraping.timeout:10000}")
       private int timeout;
       
       @Value("${factfeed.scraping.user-agent}")
       private String userAgent;

       @Override
       public List<Article> scrape(NewsSource source) {
           List<Article> articles = new ArrayList<>();
           
           for (CategoryUrl categoryUrl : source.getCategoryUrls()) {
               try {
                   articles.addAll(scrapeCategory(source, categoryUrl));
                   Thread.sleep(delayBetweenRequests); // Rate limiting
               } catch (Exception e) {
                   log.error("Failed to scrape category {} from {}: {}", 
                           categoryUrl.getCategory(), source.getName(), e.getMessage());
               }
           }
           
           return articles;
       }

       private List<Article> scrapeCategory(NewsSource source, CategoryUrl categoryUrl) {
           try {
               Document categoryPage = Jsoup.connect(categoryUrl.getUrl())
                   .userAgent(userAgent)
                   .timeout(timeout)
                   .get();

               List<Element> articleLinks = categoryPage.select(source.getArticleSelector());
               
               return articleLinks.stream()
                   .limit(10) // Limit per category to avoid overwhelming
                   .map(link -> scrapeArticle(source, link))
                   .filter(Objects::nonNull)
                   .collect(Collectors.toList());
                   
           } catch (IOException e) {
               log.error("Failed to connect to category page: {}", categoryUrl.getUrl());
               return Collections.emptyList();
           }
       }

       private Article scrapeArticle(NewsSource source, Element linkElement) {
           try {
               String articleUrl = linkElement.absUrl("href");
               
               // Check if we already have this article
               if (articleRepository.existsByUrl(articleUrl)) {
                   return null;
               }

               Document articlePage = Jsoup.connect(articleUrl)
                   .userAgent(userAgent)
                   .timeout(timeout)
                   .get();

               String title = extractTitle(articlePage, source.getTitleSelector());
               String content = extractContent(articlePage, source.getContentSelector());
               LocalDateTime publishedAt = extractPublishedDate(articlePage, source.getPublishedSelector());

               if (title.isEmpty() || content.isEmpty()) {
                   log.warn("Empty title or content for article: {}", articleUrl);
                   return null;
               }

               return Article.builder()
                   .title(title)
                   .content(content)
                   .url(articleUrl)
                   .sourceName(source.getName())
                   .publishedAt(publishedAt)
                   .scrapedAt(LocalDateTime.now())
                   .status("NEW")
                   .build();

           } catch (Exception e) {
               log.error("Failed to scrape article: {}", e.getMessage());
               return null;
           }
       }

       private String extractTitle(Document doc, String selector) {
           Element titleElement = doc.selectFirst(selector);
           return titleElement != null ? titleElement.text().trim() : "";
       }

       private String extractContent(Document doc, String selector) {
           Elements contentElements = doc.select(selector);
           return contentElements.stream()
               .map(Element::text)
               .collect(Collectors.joining("\n"))
               .trim();
       }

       private LocalDateTime extractPublishedDate(Document doc, String selector) {
           Element dateElement = doc.selectFirst(selector);
           if (dateElement != null) {
               try {
                   String dateText = dateElement.attr("datetime");
                   if (dateText.isEmpty()) {
                       dateText = dateElement.text();
                   }
                   // Parse Bangla date formats - you'll need to implement this
                   return parseBanglaDate(dateText);
               } catch (Exception e) {
                   log.warn("Failed to parse date: {}", dateElement.text());
               }
           }
           return LocalDateTime.now(); // Fallback to current time
       }
   ```
